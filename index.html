プロフィール

「子どもたちのために、より良い世界を作る」——
経営者と、経営者を支える士業の力を、AIでブーストする。

「資格の大原」で税理士講座・会計学の講師として、大学生から社会人まで幅広く会計を教えたのち、税理士法人ベリーベストへ。法人部門の責任者を経て経営企画室の室長として、採用・評価・教育・営業・情報発信を統括。スタッフ5名規模から100名規模への成長過程を、現場の最前線で経験しました。

現在は、Claude Code や freee MCP を活用した会計事務所・士業向けの生成AI活用を、セミナー・研修・講演の形でお伝えしています。弥生会計のAIセミナーでは350名超を集客し、終了10分で約100件のアンケートが寄せられるなど、現場の高い関心を実感しています。

並行して、現役の不動産投資家（投資歴7年・戸建8戸・アパート2棟）として資産税・事業承継・融資・補助金・税務調査対策にも精通。理論だけでなく、自分自身が当事者として経験してきたことが強みです。
